Run `tetris.exe` for the online version.
If the online version fails due to some JSON parsing error, you should probably run the offline version (`tetris_offline.exe`).

The offline version will work forver, the online version only works if I decide to keep the server running :)

You MUST have libcurl-x64.dll and libstdc++-6.dll in the same directory as the .exe file for these to work.
